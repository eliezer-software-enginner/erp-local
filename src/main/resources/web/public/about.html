Sobre a Aplicação

Esta é uma aplicação JavaFX baseada em WebView que utiliza HTML, CSS e JavaScript para a interface.

Arquitetura

A aplicação segue uma arquitetura em camadas onde o JavaScript controla a UI e o Java fornece os serviços do sistema.

Tecnologias

  • JavaFX WebView
  • HTML5 / CSS3
  • JavaScript ES6+
  • Protocolo JWB para comunicação JS ↔ Java
